# 怎么跑起来
## 安装依赖
`npm install`  
## 运行项目
`npm run dev`  
# 怎么部署
这个老折磨人了，查了半天Nextjs的部署  
# Docker
关于Docker的安装和使用请自行百度  
1.先拉下来一个node的镜像  
2.把项目放到镜像里  
3.运行`npm install && npm run build && npm run start`
